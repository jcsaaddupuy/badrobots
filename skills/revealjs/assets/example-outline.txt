# Slide 1
## What is Reveal.js?
- Open-source HTML presentation framework
- Works in any modern web browser
- Supports Markdown, code highlighting, and more
- Easy to share and present

# Slide 2
## Key Features
- Horizontal and vertical slides
- Markdown support
- Code syntax highlighting
- Speaker notes
- PDF export
- Responsive design
